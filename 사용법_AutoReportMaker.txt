1.요청문서(워드) : 
- 제목에 맞는 문서 생성
- 이메일 그대로 복붙

2.킥오프문서(워드) : 
- 제목에 맞는 문서 생성
- 프로젝트명/프로젝트 일자에 맞게 내용물 수정 필요(스크립트 필요)

3.결과문서(워드)
- 제목에 맞는 문서 생성
- 프로젝트명/프로젝트 일자에 맞게 내용물 수정 필요(스크립트 필요)
- 테스트 결과가 필요하므로, 체크리스트 문서의 'TEST REPORT' 정보를 가져와서 반영(스크립트 필요)

4.기타 참고문서 가져오기
- 쉐어포인트 내 팀 작업 폴더 경로를 입력
- 해당 경로에 있는 작업물을 복사해옴

위 작업들을 사용자가 버튼을 눌러 개별/통합 진행할 수 있음
그 결과는 하나의 폴더(목적지)에 저장되도록 설정
